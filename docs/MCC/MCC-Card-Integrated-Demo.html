Integration Details
Component Structure:
The MCC card includes a 25×25px rotating world sphere in the top-left corner. The sphere color matches the skill type (cyan for design/code, magenta for writing, orange for marketing, etc.). The sphere rotates rightward continuously, showing a 23° axial tilt. No equator line, single unified color, minimal aesthetic.

Usage:
Import MCCCardIntegrated and pass props for title, creator, rating, amount, tags, etc. The sphere is automatically rendered inside the card header. No additional configuration needed.

Responsive: Card scales from 320px (mobile) to full width. Sphere stays 25×25px for crisp rendering at all sizes.
